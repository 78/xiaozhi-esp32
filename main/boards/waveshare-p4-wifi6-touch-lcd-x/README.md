# Waveshare ESP32-P4-WIFI6-Touch-LCD-X


[ESP32-P4-WIFI6-Touch-LCD-X](https://www.waveshare.com/esp32-p4-wifi6-touch-lcd-7-8-10.1.htm) This series of products is a dual-core plus single-core RISC-V high-performance development board based on the ESP32-P4 chip. It is equipped with 7/8/10.1-inch IPS touch screens and supports a variety of human-computer interaction interfaces, including MIPI-CSI (integrated image signal processor ISP). It also supports USB 2.0 OTG HS to achieve high-speed connection. The ESP32-P4 chip integrates digital signature peripherals and dedicated key management units to ensure the security of data and operations. It is specifically designed for high-performance and high-security applications, fully meeting the higher requirements of embedded applications for human-machine interface support, edge computing capabilities, and IO connection characteristics.
