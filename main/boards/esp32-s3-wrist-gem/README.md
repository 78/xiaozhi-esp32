# 二、编译配置命令

**配置编译目标为 ESP32S3：**

```bash
idf.py set-target esp32s3
```

**打开 menuconfig：**

```bash
idf.py menuconfig
```

**选择板子：**

```
Xiaozhi Assistant -> Board Type -> 腕宝助手
```

**编译：**

```bash
idf.py build
```

**合并固件：**
```bash
idf.py merge-bin -o merged-xiaozhi-wg.bin -f raw
```
